Goomba Bot Command List

Please note all prefixes (+) are configurable

System Commands

Name Description Usage
+conf Define per-server configuration. +conf <set|get|reset|list|remove> [key:string] [value:string] [...]
+info Shows info on the bot. +info
+invite Displays the join server link of the bot. +invite
+ping It... like... pings. Then pongs. And it's not Ping Pong. +ping
+stats Provides some stats about the bot. +stats

Fun Commands

Name Description Usage
+8ball Magic 8-Ball, does exactly what the toy does. (Results may vary) +(8ball|8|magic|magic8|mirror) <query:str>
+achievement Sends a Minecraft Achievement image. +(achievement|mca) <title:str{1,22}>|<subtitle:str{1,22}>
+ascii Creates an ASCII banner from the string you supply +ascii <banner:str>
+banner Creates a big banner from the string you supply +banner <banner:str>
+binary Converts text to binary and vice-versa. +(binary|bin) <text_to_convert:str>
